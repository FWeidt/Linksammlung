# Linksammlung

## Was soll sie können:

- hinzufügen neuer Links aus der Webseite heraus
- keine Authorisierung nötig
- Kategorien
- Webseiten Vorschau

## Datenmodel

```json
{
    "title": "[String]",        //Titel
    "href": "[String]",         //Der eigentliche link 
    "category": "[String]",     //Kategorie
    "desc": "[String]",         //Beschreibung
    "favicon": "[String]",      //Link zum Favicon
    "online": "[Boolean]"       //
}
```
